## ⚠️ PENTING: Cara Clone Repository Ini

Project ini menggunakan **Git LFS** untuk menyimpan file model AI yang besar.
Agar file terdownload dengan sempurna, kamu **DAPAT** mengikuti langkah langkah dibawah ini

**Langkah-langkah:**

1.  **Install Git LFS**
    * **Windows:** Download & install dari [git-lfs.com](https://git-lfs.com)
    * **Linux/Ubuntu:** Jalankan perintah `sudo apt install git-lfs`
    * **Mac:** Jalankan perintah `brew install git-lfs`

2.  **Aktifkan LFS**
    Buka terminal dan jalankan:
    ```bash
    git lfs install
    ```

3.  **Clone Repository**: 
    * **Windows:** ```cmd
      1. set GIT_LFS_SKIP_SMUDGE=1
      2. git clone https://github.com/Gitkamal123/Plant-Check-Disease.git
      3. cd Plant-Check-Disease
      4. git lfs pull
      ```

    * **Linux/Mac/Git Bash:** ```bash
      1. GIT_LFS_SKIP_SMUDGE=1 git clone https://github.com/Gitkamal123/Plant-Check-Disease.git
      2. cd Plant-Check-Disease
      3. git lfs pull
      ```
    

> *Ikuti langkah diatas agar dapat mendownload secara sempurna.*
